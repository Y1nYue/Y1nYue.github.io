这是我用 Hexo + GitHub Pages 搭建的第一篇Blog。做一个简单的学习和搭建记录。

什么是 Hexo？

Hexo 是一款快速、简洁、高效的静态博客框架，主要特点包括：

  • 基于 Node.js，渲染速度快；
  • 使用 Markdown 编写文章；
  • 提供一键部署命令，支持 GitHub Pages 等多种部署方式；
  • 拥有丰富的主题与插件生态，能够轻松集成各种功能。

操作环境与前置

  • 操作系统：macOS
  • 终端环境：zsh + Oh My Zsh
  • 包管理器：Homebrew

前置：

  • 已注册 GitHub 账号；
  • 本机已安装并配置 Git；
  • 已在 GitHub 上配置好本机的 SSH Key。
